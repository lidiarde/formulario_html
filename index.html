Questionário sobre formulários HTML


a) Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

Resposta: O atributo “placeholder”, neste contexto, serve para indicar o tipo de valor (uma palavra ou uma pequena frase), o qual se espera que o usuário preencha no campo de texto do formulário.

b) Na tag <input> com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

Resposta: O atributo “checked”, neste contexto, serve para marcar ou desmarcar o referido botão “radio”.

c) Na tag <input> para que serve o atributo "value"?

Resposta: O atributo “value” serve para especificar o valor de um elemento <input>. Para tipos diferentes de <input>, o atributo “value” é usado de maneiras diferentes. No caso de “button”, “reset” e “submit”, o “value” serve para definir o texto que aparecerá no botão. No caso de “text”, “password” e “hidden”, “value” determina o valor inicial padrão para o tipo de <input>. No caso de “checkbox”, “radio” e “image”, “value” define o valor associado com a informação fornecida pelo usuário, valor que também será o enviado quando da conclusão do formulário.

d) Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

Resposta: Para agrupar botões de rádio e permitir a seleção de apenas uma opção por vez, deve-se utilizar o atributo “name”. Para tanto, todos os botões de “rádio” que se deseja agrupar devem possuir o mesmo valor associado ao atributo “name”, ou seja, cada grupo deve “possuir seu próprio nome”.

e) Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

Resposta: A tag <label> define o rótulo de vários elementos. Sua aplicação beneficia usuários de softwares de leitura de tela, fazendo a leitura dos rótulos. Beneficia também usuários com dificuldade de clicar em espaços pequenos. Por sua vez, o atributo "for", quando utilizado em conjunto com a tag <label>, serve para especificar a qual elemento do formulário o rótulo <label> deve estar associado. O valor do atributo "for" deve corresponder ao valor do atributo "id" do elemento que se deseja associar ao rótulo.

f) Na tag <option> para que serve o atributo "selected"?

Resposta: O atributo “selected” serve para pré-selecionar uma, dentre as várias opções ofertadas pela tag <option>. A opção pré-selecionada aparecerá destacada ao carregar a página web.

g) Nas tags de formulário para que serve o atributo "required"?

Resposta: O atributo “required” serve para especificar um campo <input> de entrada obrigatória antes do envio do formulário.

h) Na tag <form> para que serve o atributo "action"?

Resposta: O atributo “action” serve para especificar para onde os dados do formulário deverão ser enviados quando o usuário completá-lo.

i) Na tag <form> para que serve o atributo "method"? Quais são os valores possíveis desse atributo?

Resposta: O atributo “method” serve para determinar como os dados fornecidos pelo usuário, no formulário, devem ser enviados. Para tanto, os valores possíveis para “method” são “get” (method=“get”), no qual os dados são enviados como variáveis URL, e “post” (method=“post”), no qual os dados são enviados como uma transação HTTP. No caso do envio por “get”, ele deve ser preterido em caso de informações sensíveis, além de possuir limitação quanto ao tamanho do URL (3000 caracteres). No caso de envio por “post”, não há limitação de tamanho e há maior proteção quanto à privacidade dos dados enviados.

j) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

Resposta: Para que o valor digitado no formulário não apareça na barra de endereços do navegador, deve-se utilizar o atributo “method” com o valor “post” (method=“post”).

k) Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

Resposta: No formulário que será utilizado para o envio de arquivos, deve-se utilizar o valor “post” para o atributo “method” (method=“post”), haja vista a inexistência de limitações quanto ao tamanho dos dados enviados.

l) O que as tags <fieldset> e <legend> produzem?

Resposta: A tag <fielset> é utilizada para agrupar elementos em um formulário, construindo uma caixa ao seu redor. A tag <legend>, por sua vez, define a legenda dos elementos agrupados pela tag <fieldset>.
